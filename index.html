SaveMeLapse by xiofurry

Full-fledged easy to use timelapse-saver!

It's so easy to use, you just need to press one button!

Dependencies

Windows :( ... I will maybe add support for Mac and Linux

Install

Copy the program to the destination folder that you will create to save your .png's. After saving them, you could use VirtualDub to make an awesome timelapse.
